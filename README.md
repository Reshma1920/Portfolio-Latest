# Premium Interactive UX Portfolio

This is a **premium interactive UX portfolio** built with **HTML**, **CSS**, **JavaScript**, and **GSAP** animations. It is a high-end, interactive experience focused on user experience, motion, and micro-interactions—layered motion, rich transitions, and polished detail throughout.

## Highlights

- Crafted for immersive, performance-conscious interactions
- Layered motion and scroll-driven effects powered by GSAP
- Semantic markup and maintainable structure for a premium front-end

## Development

1. Install dependencies (when using a package manager for tooling or GSAP via npm).
2. Serve the project locally with your preferred static server or build tooling.
3. Add environment-specific secrets to `.env` (not committed) if your workflow requires them.
